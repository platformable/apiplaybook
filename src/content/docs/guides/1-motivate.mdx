In the Motivate stage, we are going to help you motivate your colleagues, decision-makers and blockers to get on board with the potential of API Governance!

<Aside>
  We prefer to call this API acceleration or "frictionless governance" to avoid
  it sounding too bureaucratic. You can work on API governance without actually
  calling it API governance.
</Aside>

<Aside type="tip">
  Other names for API governance that you can use when talking with your peers
  could include: - API value analysis - Frictionless automated API management -
  API acceleration
</Aside>

At the motivate stage you will create:

- A clear roadmap of what you can accomplish with clearly defined revenue goals
- Metrics report describing exactly where you are
- A clear overview of what you currently have in place and where the gaps are
- An API inventory of your portfolio and an understanding of which APIs are highest value

By the end of this phase you will be able to:

- Encourage action
- Open budgets
- Be invited in to discussions on major projects and AI implementations
